how it works

there are 2 main pages you can use, /random and /api
/random gets a random file and displays its path and contents
/api gets a random file and returns only its url to be used in some kind of rest api
optionally, you can supply the name of a valid directory (i.e. /random/floppa) to only get files from it
valid directory names can be found on the randomcat index page or via /directories
